Reads a message from the specified pipe file descriptor.

This function reads a message from the specified pipe file descriptor. The message is expected to be sent in chunks, where each chunk consists of a size followed by the actual data. The function reads the number of chunks first, then reads each chunk and concatenates them to form the complete message.

Parameters
readPipeFdThe file descriptor for the read end of the pipe.
debugFileThe name of the debug file for logging.
Returns
The complete message read from the pipe.
