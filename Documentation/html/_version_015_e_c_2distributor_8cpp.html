Reads incoming file paths from the server over the current pipe and adds them to the client's file list.

This function continuously reads messages from a specified pipe file descriptor. Each message represents a file name to be added to the client's file list. The function terminates when an empty message (indicating a "DONE" signal) is received.

Parameters
clientReference to the Client object to which the files will be added.
readPipeFdFile descriptor for the pipe from which messages are read.
clientIdxIndex of the client, used for logging purposes.
